## Setup
- Add `cookie-universal-nuxt` dependency using yarn or npm to your project
- Add `cookie-universal-nuxt` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    'cookie-universal-nuxt',

    // With options
    ['cookie-universal-nuxt', { /* module options */ }],
 ]
}
```
